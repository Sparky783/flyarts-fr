Random Player

Préambule

Petit projet personnel, réalisé en vue de d'améliorer mes compétences et mes connaissances en développmeent logiciel sur le framework .NET. Cette petite application a pour objectif d'aider l'utilisateur à choisir un film à regarder parmis une liste de film disponible.

Fonctionnalités

Ce logiciel permet de lancer un fichier au hasard.
Voici les fonctionnalités de l'outil:

  • Sélection automatique d'un fichier au hasard parmis une liste de fichier
  • Sélection multiple de dossier pour lister les fichiers
  • Filtre sur les fichiers parmis les types suivants: Tous, Vidéo, Image, Musique
  • Champ de recherche pour filter les fichiers par mots clés
  • Option de lancement automatique du fichier sélectionné
  • Option de recherche dans les sous-dossiers
  • Outil de renommage du fichier sélectionné
  • Outil de suppression du fichier sélectionné
  • Bouton pour sélectionner un autre fichier au hasard
  • Bouton pour revenir au fichier sélectionné précédant
  • Bouton d'actualisation de la sélection du fichier
  • Sauvegarde automatique des préférences utilisateur
  • Thème d'interface changeable entre clair et sombre

Téléchargement

Ce logiciel est portable et compatible avec les environements Windoaws uniquement.
